Calculadora de Combustível - Android

Este é um projeto inicial desenvolvido para praticar os fundamentos de desenvolvimento Android, com foco na criação de layouts e manipulação de componentes de interface.

📸 Demonstração

📋 Descrição do Projeto
O aplicativo tem como objetivo calcular o consumo de combustível com base na distância percorrida e no preço.

Nota de Desenvolvimento: Conforme orientação pedagógica (🏫 Escola Nova Era Tech), esta versão apresenta uma interface inicial simplificada, focada na estrutura funcional básica e no aprendizado de componentes de entrada de dados (EditText e TextView), antes da implementação de estilos avançados ou lógica complexa.

🚀 Tecnologias Utilizadas
Linguagem: Kotlin / Java

IDE: Android Studio

Layout: XML (ConstraintLayout)
